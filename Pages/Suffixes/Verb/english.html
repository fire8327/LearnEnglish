Suffix Description Examples
-ate a) Action (added to nouns and adjectives)
b) Transform in thing shown by root vowel		 Vaccinate ← vaccine
Granulate ← granule
-ize a) Usage/utilization (added to nouns)
b) Quality/state (added to adjectives)		 Utilize ← utility, Organize ← organ, Winterize ← winter, Theorize ← theory
Nationalize ← national
-ify Action/quality or state (added to nouns or adjectives) Beautify ← beauty, Falsify ← false, Intensify ← intense, Simplify ← simple
-en a) Usage (added to nouns)
b) State (added to adjectives)		 Threaten ← threat
Darken ← dark, Harden ← hard
