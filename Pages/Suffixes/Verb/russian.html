Suffix Description Examples
-ate a) Действие (добавляется к существительным и прилагательным)
b) Преобразование в вещь, обозначенную корневым гласным		 Vaccinate ← vaccine (вакцинировать ← вакцина)
Granulate ← granule (гранулировать ← гранула)
-ize a) Использование/применение (добавляется к существительным)
b) Качество/состояние (добавляется к прилагательным)		 Utilize ← utility (использовать ← польза), Organize ← organ (организовать ← орган), Winterize ← winter (подготовить к зиме ← зима), Theorize ← theory (теоретизировать ← теория)
Nationalize ← national (национализировать ← национальный)
-ify Действие/качество или состояние (добавляется к существительным или прилагательным) Beautify ← beauty (украшать ← красота), Falsify ← false (фальсифицировать ← ложный), Intensify ← intense (усиливать ← интенсивный), Simplify ← simple (упрощать ← простой)
-en a) Использование (добавляется к существительным)
b) Состояние (добавляется к прилагательным)		 Threaten ← threat (угрожать ← угроза)
Darken ← dark (затемнять ← темный), Harden ← hard (затвердевать ← твердый)
